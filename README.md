## ¿COMO CORRER EL PROYECTO?
Para correr el proyecto es necesario primero, instalar dependencias, usando: 
```bash
npm install
```

Luego de esto, podremos usar alguno de los siguientes comandos, siendo el mas comun entre desarrolladores ( segun mi experiencia ) el npm, para ya poder correr nuestro proyecto, aunque si no deseas realizar estos pasos a seguir, puedes ver la pagina ya desplegada en : 


```bash
https://bekind-frontend.vercel.app/
```
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
## BEKIND - PROYECTO 

Para el desarrollo de este proyecto me tome la libertad de meter mano y modificar algunas cosas del diseño ya establecido, mejorando asi la experiencia del usuario como tambien mejorar algunos elementos visuales.
Para agilizar este proceso decidi usar Tailwind, herramienta que uso mucho en cada proyecto que toco y que pueda usarlo, por su rapidez y facilidad de usar, ahorra mucho tiempo en la maquetacion. 

Asi como modifique algunas cosas del diseño, tambien agregue pantallas nuevas y extras a esta demo. Independientemente de ser aceptada o no la prueba tecnica, decidi darle mi toque personal y agregar lo que consideraba un plus a lo que se nos pidio.

Tecnologias usadas:

- React
- NextJS
- TS
- TailwindCSS

Entre lo que pude, me hubiera gustado terminar antes del tiempo limite el responsive en dispositivos mobiles. ( Recordatorio a un futuro, independientemente del resultado, hacer esta modificacion al proyecto )
